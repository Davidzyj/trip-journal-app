# Trip Journal

Trip Journal is an offline iPhone app for recording everything around a trip: packing preparation, transportation, places visited, and personal notes.

## Product

- App names: Trip Journal / 途记
- Version: 1.0.0
- Bundle ID: `com.zhouyajie.tripjournal`
- Platforms: iPhone only
- Network: none in the app runtime, except user-initiated links to privacy and support pages
- Support email: jay212315@gmail.com
- Data: local-only SwiftData storage
